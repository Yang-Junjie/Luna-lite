# Luna Lite

Luna Lite is a lightweight version of Luna Engine.

It will be written entirely by me, without the use of AI agents.

It will be meticulously refined to achieve extremely high readability and performance, and will reach the stage for release.

It will be written entirely by me, and will include:
- a software rasterization renderer,
- a path tracing renderer on the GPU,
- a rasterization renderer with a graphics API (Vulkan or OpenGL),
- a 2D physics engine.
